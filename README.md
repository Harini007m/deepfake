## 🔍 Overview

Deepfake videos generated by GANs and face-swap models often fail to reproduce the natural, irregular blinking pattern of real humans. This system exploits that weakness by:

1. Detecting 468 facial landmarks per frame using **MediaPipe Face Mesh**
2. Computing the **Eye Aspect Ratio (EAR)** from 6 key points per eye
3. Detecting **blink events** (onset → closure → reopening)
4. Extracting **behavioural metrics** (BPM, duration, symmetry, variability)
5. Comparing those metrics against **published human norms** to compute a suspicion score

| Advantage | Detail |
|-----------|--------|
| 🚀 No training required | Fully geometric & statistical |
| 💻 CPU-only | Runs on any laptop |
| 🔬 Interpretable | Every score is explained |
| 📦 Modular | 8 independent, reusable components |
| 📊 Dataset-ready | Supports Celeb-DF, UADFV layouts |

## 📦 Installation

### Prerequisites
- Python **3.9 or higher**
- pip

### Steps

```bash
# 1. Clone the repository
git clone https://github.com/your-username/deepfake-blink-detector.git
cd deepfake-blink-detector

# 2. (Recommended) Create a virtual environment
python -m venv venv
venv\Scripts\activate      # Windows
# source venv/bin/activate  # Linux / macOS

# 3. Install dependencies
pip install -r requirements.txt
```

## 🔬 Detection Logic

### Eye Aspect Ratio (EAR)

$$EAR = \frac{\|p_2 - p_6\| + \|p_3 - p_5\|}{2 \times \|p_1 - p_4\|}$$

- When the eye is **open** → EAR ≈ 0.30–0.40
- When the eye **blinks** → EAR drops below **0.22**
- A blink is confirmed after **≥ 2 consecutive frames** below threshold

### Suspicion Score

Each feature is assigned a sub-score (0 = normal → 1 = highly anomalous), then combined into a weighted total (×100):

| Feature | Weight | Normal Human Range |
|---------|--------|--------------------|
| Blinks per minute | **30%** | 8 – 21 BPM |
| Mean blink duration | **20%** | 100 – 400 ms |
| Eye symmetry diff | **20%** | < 0.25 |
| Blink variability (CoV) | **20%** | < 0.50 |
| Total blink count | **10%** | Expected ≥ min BPM |

### Verdict Thresholds

| Suspicion Score | Verdict |
|----------------|---------|
| **0 – 29** | ✅ **REAL** |
| **30 – 59** | ⚠️ **SUSPICIOUS** |
| **60 – 100** | 🚨 **LIKELY DEEPFAKE** |

## ⚙️ Configuration

All tunable parameters live in **`config.py`**:

```python
# ── EAR / Blink thresholds ──────────────────────────────────────────────
EAR_THRESHOLD          = 0.22   # EAR below this → eye is "closed"
BLINK_CONSEC_FRAMES    = 2      # consecutive closed frames to confirm a blink
EAR_SMOOTHING_WINDOW   = 3      # rolling-average window for EAR signal

# ── Normal human blink ranges ───────────────────────────────────────────
NORMAL_BPM_MIN         = 8.0    # blinks per minute lower bound
NORMAL_BPM_MAX         = 21.0   # blinks per minute upper bound
NORMAL_BLINK_DUR_MIN   = 0.10   # seconds
NORMAL_BLINK_DUR_MAX   = 0.40   # seconds
NORMAL_SYMMETRY_MAX    = 0.25   # max mean |EAR_L – EAR_R|
NORMAL_VARIABILITY_MAX = 0.50   # max coefficient of variation for IBI

# ── Scoring weights (must sum to 1.0) ───────────────────────────────────
SCORE_WEIGHTS = {
    "bpm":         0.30,
    "duration":    0.20,
    "symmetry":    0.20,
    "variability": 0.20,
    "count":       0.10,
}
```
